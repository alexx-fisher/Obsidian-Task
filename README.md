# Obsidian Task

Менеджер задач и проектов с красивым тёмным дизайном.

## Как задеплоить на Vercel

### Шаг 1 — Загрузи на GitHub
1. Зайди на github.com → нажми "New repository"
2. Назови репозиторий `obsidian-task`
3. Нажми "Create repository"
4. На следующей странице нажми "uploading an existing file"
5. Перетащи ВСЕ файлы из этой папки в браузер
6. Нажми "Commit changes"

### Шаг 2 — Задеплой на Vercel
1. Зайди на vercel.com
2. Нажми "Add New Project"
3. Выбери репозиторий `obsidian-task`
4. Нажми "Deploy"
5. Готово! Через минуту получишь ссылку на приложение 🎉

## Что умеет приложение
- ✅ Создавать и удалять проекты
- ✅ Добавлять задачи с приоритетом
- ✅ Отмечать задачи как выполненные
- ✅ Сохранять всё в браузере (localStorage)
- ✅ Работает на мобильном
